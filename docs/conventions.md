# Conventions

## Naming

Assuming the resource was named `FooBar`.

|              | Driver                 | Script                 |
| ------------ | ---------------------- | ---------------------- |
| Package      | beemo-driver-foo-bar   | beemo-script-foo-bar   |
| CLI Argument | foo-bar                | foo-bar                |
| Event Name   | foo-bar                | foo-bar                |
| File Path    | configs/fooBar.{js,ts} | scripts/FooBar.{js,ts} |
